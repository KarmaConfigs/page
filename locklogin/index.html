LockLogin modules

LockLogin SQL

LockLogin SQL module allows you to enable MySQL on your server.

This is not utile at all, as LockLogin works perfectly without MySQL, even in BungeeCord, MySQL is
not a dependency at all. MySQL is actually required when using multiple BungeeCord instances, in where the player
account should be visible by all the bungeecord instances, and not only the one the player used to connect the server.

LockLogin Premium

LockLogin Premium ( bungeecord only ) enables /premium and /offline on your network.

If you want the players to auto-login if they are premium, this is for you. Please note auto-login
does not mean bypass 2fa or pin feature, it will only skip the password phase
and never will.

LockLogin Mailer

LockLogin Mailer allows you to configure a password recovery system.

While using this module, your players will be able to use $setmail command to set an email address so if they
lose the password, they will be able to run $recovery command, to recover their accounts without the need of
removing their account.

LockLogin Manager

LockLogin Manager allows you to update LockLogin.

This module is downloaded by LockLogin automatically, always, on each plugin start. Anyway,
if for some reason it doesn't you can download it and install it as any other module. LockLogin Manager
also includes the $helpme command, which contains all the registered commands with their description.
